# MEDP33100 - Final Project: Public Archive

## Live Demo

https://final-project-nafisa-islam.onrender.com

## Project Overview

- My project focuses on people’s high school memories or memories from any type of school experience. It does not matter what kind of memory it is; it could be funny, embarrassing, emotional, or simply a moment worth remembering. The goal of this project is to create a space where people can reflect on meaningful school experiences, share their own memories, and read about the experiences of others. As part of this project, I also included one of my own school memory zines so viewers can learn a little about my experience while contributing their own stories and memories.

## Endpoints

- I used 2 endpoints such as:
                            - Get/entries, this endpoint gets all the saved school memories from the database and shows them on the Memory Wall.

                            - Post/entries, this endpoint lets users add a new school memory. It saves the memory text and sticky note color to the database.
