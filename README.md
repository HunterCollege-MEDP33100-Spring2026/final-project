## Project Overview: 
"Messages in a Bottle" is an interactive website where users can anonymously leave messages, memories, thoughts, or feelings for strangers to discover later.
This idea is inspired by the messages that people used to send in times past. For instance, in antiquity when someone wished to communicate from one side of the world to the other, people would write messages, letters, or thoughts on paper, place them inside a bottle, and cast it into the sea, in the hope that it might eventually reach its intended destination. But today with technology at our hands. Users write something personal, send it into the archive, and other people can later discover and read those messages. The goal of the project is to create emotional connection, storytelling, and a shared space for people to express themselves.

## Endpoints

GET /entries
Gets all messages from the database and displays them in the archive.
POST /entries
Creates and saves a new message.
GET /entries/
Gets one specific message.
PUT /entries/
Updates an existing message.
DELETE /entries/
Deletes a message.
